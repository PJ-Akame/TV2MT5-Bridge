# extras / MQL5

**MQL5 言語**（`.mq5` インジケーター／EA 等）の補助ツールを、**ツールごとのサブディレクトリ** に置きます。

## ディレクトリ構成

| ディレクトリ | 内容 |
|--------------|------|
| `MarketHoursDisplay/` | 市場時間・注意時間の表示用インジケーター |

新規ツールは `extras/MQL5/<ツール名>/` を追加し、その中に `.mq5`・専用 `README.md` などを格納してください。

## プロジェクト本体（Python）

自動売買・Webhook 連携は **リポジトリ直下の `MQL5\`**（Python パッケージ）です。本 `extras\MQL5` は MQL5 ソース専用で、`config.json` の `mt5` セクションとは別レイヤです。
